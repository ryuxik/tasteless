Get started

We started Lumen because every tool we used felt louder than the work it was supposed to hold. Notifications stacked on notifications, dashboards sprouting dashboards, a settings panel for the settings panel. The industry had confused density with capability, and the people doing the actual work were paying for it in attention they could not spare.

So we built the opposite. A surface that recedes. A grid that breathes. Typography that trusts you to read. Underneath, the same speed and rigor you expect — version history, granular permissions, an API that does not fight you — but presented so that none of it asks for your attention until you ask for it first.
